HTML Element

Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.

Title Element

The title tag is an HTML title element critical to both SEO and user experience that is used to briefly and accurately describes the topic and theme of an online document.

Heading Element

The heading elements are H1, H2, H3, H4, H5, and H6 with H1 being the highest (or most important) level and H6 the least. For example:

Body Element

The tag defines the document's body. The element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

Heading Element (visually)

Paragraph Element

The HTML

element represents a paragraph of text. Paragraphs are usually represented in visual media as blocks of text that are separated from adjacent blocks by vertical blank space and/or first-line indentation.

Comment Element

Comments are things you write in HTML that won't show up in real website.

Image Element

The tag defines an image in an HTML page. The tag has two required attributes: src and alt.

List Element
